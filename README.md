A React-based web application for managing and displaying employee information and organizational structure for Pixell River Financial.

## Project Overview

This application provides two main features:
1. **Employee Directory** - Searchable list of company employees
2. **Organization Structure** - Interactive display of leadership roles with expandable descriptions

## Features

- **Routing**: Navigate between Employees and Organization pages
- **Search Functionality**: Filter employees by name, position, or department
- **Expandable Content**: Click on organization roles to view detailed descriptions
- **Responsive Design**: Clean and simple UI that works across devices
